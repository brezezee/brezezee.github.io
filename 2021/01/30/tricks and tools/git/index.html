提交PR流程

  1. fork源库到自己的仓库
  2. git clone 这个fork下来的仓库
  3. git remote add upstream 源库链接
  4. 新建分支git checkout -b feature-xx
  5. git add .
  6. git commit -m “xxx”
  7. git push
  8. 最后去网页提交PR
